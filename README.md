# homework_repo

# Домашнее задание по курсу "Питон Анализ данных"

Этот репозиторий содержит материалы домашнего задания по курсу "Питон Анализ данных".

## Содержание

* **`HW/`**: Папка с файлами домашнего задания.
    * `HW_1.ipynb`: Основной Jupyter Notebook с выполненным домашним заданием.
    * `hotel_booking_data.csv`: Набор данных, используемый в задании.
* **`requirements.txt`**: Список всех Python-библиотек с их версиями, необходимых для воспроизведения проекта.
* **`.gitignore`**: Файл, указывающий Git, какие файлы и папки следует игнорировать (например, виртуальное окружение `venv/`).
* **`LICENSE`**: Лицензия MIT, определяющая условия использования кода в этом репозитории.

## Лицензия
