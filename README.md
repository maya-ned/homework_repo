# homework_repo

# Домашнее задание по курсу "Питон Анализ данных"

Этот репозиторий содержит материалы домашнего задания по курсу "Питон Анализ данных".

## Содержание

* **`HW/`**: Папка с файлами домашнего задания.
    * `HW_1.ipynb`: Основной Jupyter Notebook с выполненным домашним заданием.
    * `hotel_booking_data.csv`: Набор данных, используемый в задании.
* **`requirements.txt`**: Список всех Python-библиотек с их версиями, необходимых для воспроизведения проекта.
* **`.gitignore`**: Файл, указывающий Git, какие файлы и папки следует игнорировать (например, виртуальное окружение `venv/`).
* **`LICENSE`**: Лицензия MIT, определяющая условия использования кода в этом репозитории.

## Установка и запуск

Для запуска и проверки домашнего задания выполните следующие шаги:

1.  **Клонируйте репозиторий:**
    ```bash
    git clone [https://github.com/ВАШ_ЛОГИН/ВАШ_РЕПОЗИТОРИЙ.git](https://github.com/ВАШ_ЛОГИН/ВАШ_РЕПОЗИТОРИЙ.git)
    cd ВАШ_РЕПОЗИТОРИЙ
    ```
    *(Замените `ВАШ_ЛОГИН` и `ВАШ_РЕПОЗИТОРИЙ` на свои данные).*

2.  **Создайте и активируйте виртуальное окружение:**
    ```bash
    python -m venv venv
    # Для Windows:
    venv\Scripts\activate
    # Для macOS/Linux:
    source venv/bin/activate
    ```

3.  **Установите необходимые библиотеки:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Запустите Jupyter Notebook:**
    ```bash
    jupyter notebook
    ```
    После запуска Jupyter откройте файл `HW/HW_1.ipynb` в браузере.
